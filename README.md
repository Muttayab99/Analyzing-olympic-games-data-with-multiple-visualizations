# Analyzing-olympic-games-data-with-multiple-visualizations


This repository includes a comprehensive Tableau analysis of Olympic data .
---

### **Dashboard 1 - Overview**
- **Visualization Type:** Interactive dashboard
- **Purpose:** Comprehensive overview of Olympic statistics
- **Components:**
  - Key metrics (total medals, countries, athletes)
  - Top performers
  - Geographic distribution
  - Time series trends
- **Interactivity:** Filters for year, country, sport, season

### **Dashboard 2 - Deep Dive**
- **Visualization Type:** Advanced analytical dashboard
- **Purpose:** Detailed analysis with drill-down capabilities
- **Components:**
  - Athlete demographics
  - Sport-specific breakdowns
  - Comparative analysis tools
  - Custom calculated fields
- **Interactivity:** Cross-filtering, parameter controls, tooltips

---

### Visualization Techniques Used

1. **Bar Charts** - Comparing medal counts across categories
2. **Line Graphs** - Showing trends over Olympic years
3. **Geographic Maps** - Spatial distribution of medals
4. **Treemaps** - Hierarchical medal distribution
5. **Scatter Plots** - Analyzing athlete demographics
6. **Heat Maps** - Intensity of performance across dimensions
7. **Pie Charts** - Proportional medal distribution
8. **Stacked Bar Charts** - Combining multiple medal types
9. **Box Plots** - Statistical distribution of athlete characteristics
10. **Highlight Tables** - Quick identification of patterns
11. **Dashboards** - Interactive combined visualizations
12. **Calculated Fields** - Custom metrics and KPIs

### How to Use
1. Install Tableau Desktop or Tableau Reader
2. Open `storyboard.twb` in Tableau
3. Explore the 14 worksheets/dashboards
4. Use filters to focus on specific:
   - Countries
   - Olympic years
   - Sports/Events
   - Seasons (Summer/Winter)
   - Medal types
5. Hover over data points for detailed tooltips
6. Export visualizations or create custom views
